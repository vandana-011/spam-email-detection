# spam-email-detection
# Spam Email Detection using Machine Learning

## Description
This project classifies emails as Spam or Not Spam using NLP and Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF
- Naive Bayes

## Model Used
Multinomial Naive Bayes

## Accuracy
97%

## How to Run
1. Install dependencies
2. Run `spam_detection.py`

## Use Case
Used by email providers to filter spam emails automatically.
